# 📊 Retail Sales Forecasting & Demand Intelligence System

## 📌 Project Overview

This project is an end-to-end Sales Forecasting and Demand Intelligence System developed using Python and Machine Learning. It analyzes retail sales data, forecasts future sales, detects anomalies, segments products based on demand patterns, and presents insights through an interactive Streamlit dashboard.

---

## 🎯 Objectives

- Analyze historical retail sales data
- Forecast future sales using multiple forecasting models
- Detect unusual sales patterns
- Segment products based on demand behavior
- Build an interactive business dashboard
- Generate actionable business recommendations

---

## 📂 Dataset

- **Primary Dataset:** Superstore Sales Dataset (`train.csv`)
- **Secondary Dataset:** Video Game Sales Dataset (`vgsales.csv`)

---
